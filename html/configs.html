Don't know what config files are?

Config files are where you store your command data. All the commands you type in the config file will be pasted to the console in game. It's a quick way to launch your favorite settings or binds withouth much work.

This is useful if you want to save custom settings that you want to launch when opening Counter Strike.

They are written as [FileName].cfg and you can find them in:

C:\Program Files (86x)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg

Creating a cfg file

  1. Find the directory of cfg files from the link above.
  2. After navigating to the directory, create a random.txt
  3. Open and add your commands
  4. Press save as and make the extension .cfg

CS:GO useful commands

Make sure to edit the bind keys, it may interfere with your existing key bindings!

  • alias "+jumpthrow" "+jump;-attack"; alias "-jumpthrow" "-jump"; bind [KEY] "+jumpthrow";

    • Jump throw bind. Withouth this, your jumpthrow grenades will not be consistent.

  • fps_max 120

    • Sets the max framerate of CS:GO. fps_max 0 unlocks framerate limit

  • cl_radar_always_centered

    • Allows you to see all of the map in radar.

  • cl_use_opens_buy_menu 0

    • Doesn't allow you to use "use" (default E) key to open buy menu.

  • +cl_show_team_equipment

    • Allows you to see your teamate equipment.

  • cl_lefthand

    • Use left hand instead of default right

  • voice_enable 0

    • If you're sick of hearing your teamates, use this.

  • bind [KEY] "r_cleardecals"

    • After pressing the key, clears all blood splatters and residual particles which may interfere with vision.

Your launcher cfg file may look something like this

alias "+jumpthrow" "+jump;-attack"; alias "-jumpthrow" "-jump"; bind x "+jumpthrow";

fps_max 120

bind v "r_cleardecals"

cl_use_opens_buy_menu 0

How to launch them

If you won't add these cfg files to the launcher options, they will not run automatically. To do so manually in game you have to:

  1. Launch the game.
  2. Enable the Developer console in settings.
  3. Press the backtick button (`) near your (ESC) button.
  4. Type: "exec [FileName].cfg"

Launcher options

Launching cfg files on startup

You can add any cfg file to autoload when starting CS:GO by editing the launcher options.
You can find them in steam by right clicking on CS:GO and pressing properties. Then you will see called Launch Options. All that's left is to add your commands

Launch option commands

  • +exec [filename].cfg

    • Will execute your created cfg file.

  • -nojoy

    • Will remove joystick support from the game, which may free up RAM.

  • -novid

    • Skips the intro video.

  • -language

    • Sets the language for your game.

  • -tickrate 128

    • Sets the tickrate for offline matches (against bots).

    Warning! The default is 64. This may interfere with grenade practice.

  • -windowed or -fullscreen

    • Loads the game in windowed or fullscreen mode.

That's it! Now every time you start CS:GO these specific commands will be launched.

Your launch options may look something like this

-nojoy -novid +exec starting.cfg

    sv_autobunnyhopping 1

    • enables bunny hopping.

  • mp_solid_teammates 0

    • enables to walk through teamates.

  • net_graph 1

    • Shows detailed description of the game (framerate, tickrate, ping...).

  • god

    • Makes you an undying god.
